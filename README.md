## BugBountyTools
All I need to start !

## Installation

Use git command to clone repo

```bash
git clone https://github.com/madshei/BugBountyTools.git
```

## Then

```bash
cd BugBountyTools
```
```bash
chmod +x tools.sh
```
```bash
./tools.sh
```
